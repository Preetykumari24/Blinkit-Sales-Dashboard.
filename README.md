# Blinkit Sales Dashboard

## Project Overview
This Power BI dashboard analyzes Blinkit sales performance, customer ratings, item categories, outlet types, and outlet locations.

## Tools Used
- Power BI
- Excel
- DAX
- Data Modeling

## Key KPIs
- Total Sales: $1M
- Average Sales: $141
- Number of Items: 8,523
- Average Rating: 3.92

## Dashboard Insights
- Outlet sales trend by establishment year
- Sales by outlet size
- Sales by outlet location
- Item type performance
- Fat content analysis
- Outlet type comparison
